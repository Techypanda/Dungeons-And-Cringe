Dungeons & Cringe - A Object Oriented Approach
Java Application written for my Object Oriented Software Engineering unit in Curtin University

README - OOSE Assignment 2020

WHAT IS IT?: An assignment about using various design patterns to make a game using language of choice (I chose Java)

Dependencies:
    JDK 8+ (Diamond Operator is used)

How To Run:
    1) cd into Jar Build
    ---- Using Ant To Automate The Build ----
    2) run ant in terminal
    3) cd into dist
    4) java -jar OOSE-Assignment-JonathanWright.jar
    ---- Manual Run ----
    2) cd into src/com/jonathan
    3) javac Program.java into a jar file/run it manually
    4) Either run the created jar or the class file.
